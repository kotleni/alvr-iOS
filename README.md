## ALVR for iOS
Custom ALVR client implementation for iOS.

### Whats implemented?
- [x] Basic streaming
- [x] Battery state update
- [x] ARTracking (high CPU load)
     - [x] Orientation
     - [x] Position
- [ ] Render enchantments
    - [ ] Foveating encoding
    - [ ] Reprojection
