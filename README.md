## ALVR for iOS
Custom ALVR client implementation for iOS.

### Whats implemented?
- [x] Basic streaming
- [ ] Cardboard support
- [x] Battery state update
- [x] AR Tracking (high CPU/Power usage)
     - [x] Orientation
     - [x] Position
- [ ] Motion tracking (low CPU/Power usage)
    - [ ] Orientation
    - [ ] ~~Position~~ *1
- [ ] Render enchantments
    - [ ] Foveating encoding
    - [ ] Reprojection
